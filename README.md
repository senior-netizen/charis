# Charis Nutrition Website

Production-ready Next.js (App Router) starter for a nutrition and wellness practice.

## Stack
- Next.js 14 + TypeScript
- Tailwind CSS
- Framer Motion

## Local development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run start
```

## CMS-ready architecture
Business content lives in `/content` and `/data` as typed modules. Replace these with a headless CMS adapter (e.g., Sanity, Contentful, or Strapi) without changing page or component contracts.
